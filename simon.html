How to Play:

  1. Start the Game by selecting the Difficulty Level. Easy, Medium, or Hard.
  2. Watch as the the colours pulse, and try to remember the sequence.
  3. When the sequence has complete, click/touch the colours in the correct sequence.
  4. If the sequence is entered correctly, You win. If not, start again.
